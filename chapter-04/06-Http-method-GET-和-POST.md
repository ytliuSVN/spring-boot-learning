# 單元 6 - Http method - GET 和 POST

### GET

- 最常使用的 http method
- 類似於明信片的概念，傳遞的參數會被別人看見，安全性較低

![GET 參數放在 URL（query parameter）](assets/06-01-get-query-parameter.png)

參數直接放在 url 裡面，會被其他人看見

### POST

- 也是很常使用的 http method
- 類似於信封的概念，傳遞的參數不會被別人看見，安全性較高

![POST 參數放在 request body](assets/06-02-post-request-body.png)
